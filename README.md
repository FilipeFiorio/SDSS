# SDSS — Super Data Structures Simulator

Simulador visual de estruturas de dados desenvolvido em Java com a biblioteca [JSGE](https://github.com/davidbuzatto/JSGE).

## Estruturas simuladas

- **Pilha** (Stack)
- **Fila** (Queue)
- **Deque** (Double-Ended Queue)
- **Lista** (List)
- **Árvore Binária de Busca** (BST)
- **Árvore AVL**
- **Árvore Rubro-Negra** (Red-Black Tree)

## Tecnologias

- Java
- JSGE (Java Simple Game Engine)
- NetBeans IDE

## Como executar

1. faça o download do .zip ou clone o repositório:
   ```bash
   git clone https://github.com/FilipeFiorio/SDSS.git
   ```
2. Abra o projeto no **NetBeans IDE**
3. Execute com a própria IDE (f6)
 

## Download

Acesse a página de [Releases](https://github.com/FilipeFiorio/SDSS/releases) para baixar a versão mais recente do SDSS em `.jar`.

Após baixar, execute com:
```bash
java -jar SDSS.jar
```

## Sobre

Projeto acadêmico com fins educacionais, voltado ao aprendizado visual de estruturas de dados clássicas da ciência da computação.
